锯齿靶标法

Principle

在进行激光扫描重建三维模型时，不仅要求解出相机的内部参数，相机坐标系与世界坐标系转换的参数也要求解出来。锯齿靶标法利用结构光与锯齿靶相切来获取特征点的坐标值，实现相机的外参标定。

本文以旋转式激光扫描为例，图中$O_W-X_WY_WZ_W$是以旋转平台圆心为原点建立的世界坐标系，锯齿靶标放置在移动导轨上并与旋转平台垂直，导轨移动方向为$Z$轴方向。激光垂直照射在锯齿靶标上，即激光平面为$XOZ$平面。通过测量可以得出锯齿靶标相邻齿根之间的距离以及齿顶到齿根的垂直距离。若最下面一个齿根位于原点上，则锯齿靶标上位于激光平面内的所有齿根齿顶的世界坐标都可以得知。

摄像机拍摄带有激光的锯齿靶标图像，通过检测激光特征点，获取位于齿根齿顶处的像素坐标。导轨向前移动特定的距离，这些特征点的世界坐标依然是明确可知的。通过获取多组特征点数据，利用世界坐标值与像素坐标值以及相机内参，可以求得旋转矩阵与平移矩阵的最优参数，完成外参标定。

锯齿靶标法的难点在于如何检测出图像上的激光特征点，并获取齿根齿顶的坐标。激光线条可以通过Steger算法进行测量，并获得每个激光特征点的法线方向，通过法线方向的变化来检测出齿根齿顶。

Experiment

  1. 拍摄锯齿靶标图像

建议晚上拍摄会更清晰，激光中心会出现明亮的白光，或许使用滤光片也会有不错的效果。

将图像横放，是因为我在写激光检测算法时，是将图像从左到右一列一列的进行检测。为了激光线条顺序与检测特征点顺序一致，才进行横放。如果不横放，将检测出的特征点按照行数从小到大排序也可以使其一致。

  1. 检测图像白光区域

为了去除激光外的噪声点，我们将只获取图像中的白色激光区域，来更好地检测出激光线条。主要方法是将图像转换为HSV颜色空间，保留图像上的白色区域范围。

void inrange_demo(Mat& image)
{
    Mat hsv;
    Mat dst;
    cvtColor(image, hsv, COLOR_BGR2HSV);
    Mat mask;
    inRange(hsv, Scalar(0, 0, 221), Scalar(180, 30, 255), mask); 
    bitwise_and(image, image, dst, mask);
    image = dst.clone();
}

  1. 激光检测

激光检测采用Steger算法，检测出图像上的激光特征点，并计算特征点的法线方向。

class Steger {
public:
	Steger(Mat, int);
	~Steger();
    void Init();
    void FindCenterLine(vector<Point2f>&, vector<Vec4f>&);
    void FindNormal(int, int, double&, double&);
	Mat Img;
    int Threshold;
	Mat dx, dy;
	Mat dxx, dyy, dxy;
private:

};
Steger::Steger(Mat image, int thres) {
    Threshold = thres;
    Img = image;
    cvtColor(Img, Img, COLOR_BGR2GRAY);
    cout << "size of Img:" << Img.cols << "*" << Img.rows << endl;
}
Steger::~Steger() {

}
void Steger::Init() {
    Mat image;
    Img.convertTo(image, CV_32FC1);
    GaussianBlur(image, image, Size(0, 0), 6, 6);
    Mat m1, m2, m3, m4, m5;
    m1 = (Mat_<float>(3, 1) << 1, 0, -1);  //x偏导
    m2 = (Mat_<float>(1, 3) << 1, 0, -1);  //y偏导
    m3 = (Mat_<float>(3, 1) << 1, -2, 1);   //二阶x偏导
    m4 = (Mat_<float>(1, 3) << 1, -2, 1);   //二阶y偏导
    m5 = (Mat_<float>(2, 2) << 1, -1, -1, 1);   //二阶xy偏导

    filter2D(image, dx, CV_32FC1, m1);
    filter2D(image, dy, CV_32FC1, m2);
    filter2D(image, dxx, CV_32FC1, m3);
    filter2D(image, dyy, CV_32FC1, m4);
    filter2D(image, dxy, CV_32FC1, m5);
}
void Steger::FindNormal(int y, int x, double& nx, double& ny) {
    Mat hessian(2, 2, CV_32FC1);
    hessian.at<float>(0, 0) = dxx.at<float>(y, x);
    hessian.at<float>(0, 1) = dxy.at<float>(y, x);
    hessian.at<float>(1, 0) = dxy.at<float>(y, x);
    hessian.at<float>(1, 1) = dyy.at<float>(y, x);
    Mat eValue;
    Mat eVectors;
    eigen(hessian, eValue, eVectors);  //两个特征值
    //点(x,y)的法线方向可由该点 Hessian 矩阵绝对值最大的特征值所对应的特征向量(nx,ny)给出
    //cout << eValue << endl;
    if (fabs(eValue.at<float>(0, 0)) >= fabs(eValue.at<float>(1, 0)))  //fabs返回浮点数绝对值
    {
        nx = eVectors.at<float>(0, 0); //特征向量
        ny = eVectors.at<float>(0, 1);    }
    else
    {
        nx = eVectors.at<float>(1, 0);
        ny = eVectors.at<float>(1, 1);
    }
}
void Steger::FindCenterLine(vector<Point2f>& Pts, vector<Vec4f>& line) {
    for (int i = 0; i < Img.cols; i++)
    {
        for (int j = 0; j < Img.rows; j++)
        {
            if (Img.at<uchar>(j, i) >= Threshold)
            {
                double nx, ny;
                FindNormal(j, i, nx, ny);//寻找激光光条法线方向
                // 判断亚像素点是否在该像素内，该点 (x0,y0) 为光条的中心点， (px,py) 则为亚像素坐标
                double bottom = (nx * nx * dxx.at<float>(j, i) + 2 * nx * ny * dxy.at<float>(j, i) + ny * ny * dyy.at<float>(j, i));
                if (bottom != 0) {
                    double t = -(nx * dx.at<float>(j, i) + ny * dy.at<float>(j, i)) / bottom;
                    if (fabs(t * nx) <= 0.5 && fabs(t * ny) <= 0.5)
                    {
                        line.push_back(Vec4f(nx, ny, i + t * nx, j + t * ny));
                        Pts.push_back(Point2f(i + t * nx, j + t * ny));
                    }
                }
            }
        }
    }
}

为了有效检测出齿根齿顶点，我们将过滤掉法线方向过大的点。

  1. 齿根齿顶检测

通过判断法线方向的变化是否超出了阈值，来判断该点是否是一个位于齿根或齿顶的特征点。

阈值是自定义的，不同图像可能会有所差别，如果可以研究出一种自适应阈值算法，可能会更完整地检测出特征点。

vector<Point2f> LaserDetect(Mat& img) {
	int threshold = 100;
	Steger Image = Steger(img, threshold);
	Image.Init();
	vector<Point2f> Pt;
	vector<Vec4f> line;
	Image.FindCenterLine(Pt, line);
	cout << Pt.size() << endl;
	float lastK = 0.0;
	float currentK = 0.0;
	int thres = 0.3;
	for (int i = 0; i < Pt.size(); i++) {
		currentK = line[i][1] / line[i][0];
		if (fabs(currentK) > 1.5) {
			continue;
		}
		cout << currentK << endl;
		circle(img, Pt[i], 2, Scalar(255, 0, 0), -1);
		//drawaaLine(img, line[i], Pt[i], Scalar(0, 255, 255));
		if (i == 0) {
			continue;
		}
		currentK = line[i][1] / line[i][0];
		if (fabs(lastK - currentK) <= 0.25) {
			lastK = line[i][1] / line[i][0];
			continue;
		}
		else { 
			thres = fabs(lastK - currentK);
			circle(img, Pt[i], 4, Scalar(0, 0, 255), -1);
		}
		lastK = line[i][1] / line[i][0];
	}
	return Pt;
}
